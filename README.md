# Project Roble

+ 중단된 개발 프로젝트에서 작성했던 코드를 저장하기 위한 Repository입니다
+ 현재 Repository에는 본인이 작성한 코드만 등록하였으며 이에 대해서만 설명할 것을 밝힙니다

+ 디렉토리 구조
  + __Scripts__: 빌드 테스트에 사용되었던 코드를 모아놓은 디렉토리입니다
  + __Refectoring Scripts__: 보완점이 필요한 코드들을 재작성하고 있던 디렉토리입니다
